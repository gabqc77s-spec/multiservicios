# Fire Service

A simple Python script designed to sum two numbers passed as command-line arguments.

This service is intentionally created with an initial syntax error (`prnt` instead of `print`) to demonstrate auto-healing capabilities in a CI/CD pipeline or development environment.

## Usage

To run the script (after correcting the syntax error):

```bash
python main.py <number1> <number2>
```

## Example

```bash
python main.py 5 10
# Expected output: The sum of 5 and 10 is: 15
```
